[
	{
		name,
		preprocess: ( { ...params } ) => ({ data, search, hash })
            // data: {}, search: {}, hash: string
	}
]
вызов ссылки или редиректа тогда будет такой:
<Link to='name' params={{p1:v1,p2:v2}}></Link>
redirect( name, params )
